    About the Process

    The purpose of this project was to incorporate Machine Learning (ML) into a Spotify dataset that would predict a desired song that best matches a user input. We began by identifying CSV's from kaggle.com that provided various song metrics over multiple decades. From there we cleaned the data through Pandas to create a combined CSV from 1960s-2010s. Once we had the data compiled into one CSV we began identifying which song metrics would resonate most with users. We landed on Energy, Temp and Danceability as desired metrics users would most likely associate with song selection.
